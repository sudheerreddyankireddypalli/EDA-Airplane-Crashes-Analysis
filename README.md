# EDA Project: Airplane Crashes Dataset
This project explores over 5,000 airplane crashes since 1908 to uncover patterns in crash frequency and high-risk operators.

## Steps
- Data cleaning (nulls, categorical values)
- Feature engineering (date, operator, location)
- Visualization (heatmaps, bar plots, scatter plots)

## Key Insights
- Crash frequency peaked in mid-20th century.
- Certain operators had disproportionately high incidents.
- Weather and location factors influenced crash severity.
